# Constructor University — Interactive Timeline

Interactive historical timeline of Constructor University (1936–present).

## Публикация — только через браузер, ничего ставить не нужно

### Шаг 1: Аккаунт на GitHub
1. Зайди на **https://github.com** → **Sign up**
2. Введи email, пароль, username, подтверди email

### Шаг 2: Создай репозиторий
1. Нажми **+** (правый верхний угол) → **New repository**
2. Name: **constructor-timeline**
3. Оставь **Public**, галочки НЕ ставь
4. **Create repository**

### Шаг 3: Загрузи файлы
1. На странице пустого репозитория нажми **uploading an existing file**
2. Распакуй ZIP на компьютере
3. Перетащи ВСЕ файлы и папки в окно загрузки
4. Нажми **Commit changes**

⚠️ Папка `.github` скрытая! Чтобы увидеть:
- **Mac**: Cmd + Shift + .
- **Windows**: Проводник → Вид → Скрытые элементы

### Шаг 4: Включи Pages
1. В репозитории: **Settings** → **Pages**
2. Source: **GitHub Actions**
3. **Save**

### Шаг 5: Готово!
Через 2-3 минуты сайт на:
`https://ТВОЙ_USERNAME.github.io/constructor-timeline/`
